# Neuron

> AI-Powered Financial Wellness Platform

Neuron is a comprehensive financial management system designed to advocate for the user's financial well-being. It combines traditional financial tracking with advanced AI insights to provide a holistic view of your wealth, incorporating behavioral psychology and expert financial planning.

## Key Features

- **Smart Dashboard**: Real-time overview of your financial health, including net worth, recent transactions, and key metrics.
- **Transaction Management**: Aggregate and categorize transactions from multiple sources.
- **Portfolio Tracking**: Monitor investments and asset performance.
- **AI Insights**: Personalized recommendations and insights powered by artificial intelligence to help you make better financial decisions.

## Technology Stack

- **Frontend**: React (Vite, TypeScript, TailwindCSS)
- **Backend / Database**: Supabase (Planned/Implied)
- **Tools**: Lucide React, Recharts, Zustand

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd neuron
    ```

2.  Navigate to the frontend application:
    ```bash
    cd apps/web
    ```

3.  Install dependencies:
    ```bash
    npm install
    ```

4.  Start the development server:
    ```bash
    npm run dev
    ```
